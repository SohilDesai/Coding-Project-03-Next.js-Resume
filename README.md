# Next.js Resume

A single-page resume website built with Next.js App Router and Tailwind CSS.
## How to Run Locally

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

3. Open your browser and visit:

```
http://localhost:3000
```
## Features Implemented

- [x] Next.js App Router project
- [x] Resume recreated using JSX
- [x] Semantic HTML structure
- [x] Tailwind CSS utility classes
- [x] Education section
- [x] Work experience section
- [x] Skills section
- [x] Certifications section
- [x] Volunteerism section
- [x] Responsive resume layout
